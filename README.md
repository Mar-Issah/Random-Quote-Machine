# Random Quote Machine
This is a web application which displays random quotes made by famous people


## About The Project

![Random Quote Machine](https://res.cloudinary.com/dytnpjxrd/image/upload/v1617297529/My%20Website%20Projects/random_quote_machine_twdozo.png)

This app basically displays quotes by famous people. It shows the quotes as well as the authors name. It also has a tweet button which enables user to share quote on twitter.

## Getting Started
This is JavaScript Web application which runs in the browser. You need a V8 engive to run JavaScript program. For the purpose of this project I used Google Chrome browser.

### Installation

1. Clone the repo on your local machine
   ```sh
   git clone https://github.com/Mar-Issah/Random-Quote-Machine.git
   ```
2. Download and install [Visual Studio Code](https://code.visualstudio.com/)
3. Install live server extension
4. Open index.html file with live server to run the app in browser
